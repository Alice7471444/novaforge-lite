# NovaForge Lite - Usage Guide
## ⚡ How to Use Your Lightweight Cyberpunk OS

---

## 🚀 Getting Started

### First Boot
1. **Language Selection** - Choose your language
2. **Network** - Connect to WiFi (optional)
3. **User Account** - Create your username and password
4. **Done!** - Desktop loads automatically

That's it! No terminal commands needed! 🎉

---

## 🌐 Internet & Networking

### Connecting to WiFi
1. Click the network icon in the panel (bottom-right)
2. Select your WiFi network
3. Enter password and connect

### Ethernet
Simply plug in your Ethernet cable - it connects automatically!

### Bluetooth
1. Click Bluetooth icon
2. Enable Bluetooth
3. Pair your devices

---

## ⚙️ Settings

### Changing Wallpaper
1. Right-click on desktop
2. Select "Configure Desktop"
3. Choose wallpaper

### Theme Settings
1. Open **Settings**
2. Go to **Appearance**
3. Customize colors, icons, theme

### Display Settings
1. Open **Settings**
2. Go to **Display**
3. Adjust resolution, orientation, scaling

### Power Settings
1. Open **Settings**
2. Go to **Power**
3. Set sleep, display off times

---

## 📦 Applications

### Opening Apps
- Click app icon in dock
- Or press `Super` and search

### Pre-installed Apps
| App | Use |
|-----|-----|
| Firefox | Web browsing |
| Dolphin | File manager |
| VLC | Videos & music |
| Steam | Gaming |
| Settings | System settings |

### Installing Apps
Use the built-in app store or:
```bash
# Terminal (if needed)
sudo pacman -S <app-name>
```

---

## 🎮 Gaming

### Steam
1. Open **Steam** from dock
2. Login or create account
3. Download games!

### GameMode
Automatically enabled for compatible games.

### Performance
Use NOVA AI to optimize:
```bash
nova-ai-lite optimize
```

## 🤖 NOVA AI Lite

### Commands
```bash
# Get help
nova-ai-lite help

# System info
nova-ai-lite info

# Clear RAM
nova-ai-lite optimize

# Search web
nova-ai-lite search [query]

# Wallpapers
nova-ai-lite wallpaper

# Updates
nova-ai-lite update
```

## ⌨️ Keyboard Shortcuts

### Basic
| Key | Action |
|-----|--------|
| `Super` | Open app launcher |
| `Alt + F4` | Close window |
| `Alt + Tab` | Switch apps |

### Screenshots
| Key | Action |
|-----|--------|
| `Print` | Screenshot |
| `Super + Print` | Screenshot with UI |

### Virtual Desktops
| Key | Action |
|-----|--------|
| `Super + Tab` | View desktops |
| `Super + Arrow` | Switch desktop |

## 🔧 Troubleshooting

### Slow Performance
```bash
nova-ai-lite optimize
```
Or reduce desktop effects in Settings

### WiFi Issues
1. Turn off WiFi
2. Turn back on
3. Reconnect

### Display Issues
Adjust in Settings → Display

### Freezing
Press `Ctrl + Alt + Delete` to logout
