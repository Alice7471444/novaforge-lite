## ⬇️ Download & Install

### Option 1: Ready-to-Use ISO
Download the ISO from [Releases](https://github.com/Alice7471444/novaforge-lite/releases) and burn to USB:

```bash
sudo dd if=NovaForge-Lite-v1.0.0-x86_64.iso of=/dev/sdX bs=4M status=progress
sync
```

### Option 2: Build from Source
```bash
git clone https://github.com/Alice7471444/novaforge-lite.git
cd novaforge-lite
sudo ./build-lite.sh build
```
