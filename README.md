## 🎮 Built for Low-End Hardware

- **2GB RAM** optimized
- **KDE Plasma** - snappy and modern
- **Wayland** compositor
- **Auto-login** via SDDM

### Install

```bash
git clone https://github.com/Alice7471444/novaforge-lite.git
cd novaforge-lite
sudo ./build-lite.sh build
```
