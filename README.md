## 🛠️ Installation

### VMware
1. Open VMware Workstation/Fusion
2. Create new virtual machine
3. Select "I will install the operating system later"
4. Choose "Linux" → "Other Linux 6.x kernel"
5. Configure:
   - RAM: 2 GB (4 GB recommended)
   - CPU: 2 cores
   - Hard Disk: 20 GB
   - Network: NAT
6. Mount the NovaForge Lite ISO
7. Start the VM and follow installer

### Hardware
1. Create bootable USB using Rufus or Etcher
2. Boot from USB
3. Follow the graphical installer
4. Enjoy NovaForge Lite!

## 🎮 Getting Started

### First Boot
1. Welcome screen → Select language
2. Connect to WiFi (optional)
3. Create user account
4. That's it! Desktop loads automatically

### No Terminal Required
All settings accessible via GUI:
- **WiFi**: Click network icon → Select network
- **Bluetooth**: Click Bluetooth icon → Enable
- **Updates**: Settings → Updates
- **Apps**: Application launcher

## 🔧 Development

### Building from Source
```bash
# Clone repository
git clone https://github.com/Alice7471444/novaforge-os.git
cd novaforge-os

# Build Lite edition
```
